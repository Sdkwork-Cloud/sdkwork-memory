# SDKWork Memory Retrieval Component

This directory indexes the local contract for `sdkwork-memory-retrieval`.
The machine-readable authority is `component.spec.json`; canonical SDKWork
standards remain in `../../../sdkwork-specs/`.

The component owns deterministic, provider-independent candidate scoring,
rank fusion, and token-budgeted context assembly. It does not own canonical
memory persistence, provider transport, authorization, or generated SDKs.
